What is Bish-Bosh?

Bish-Bosh is a game where you count from 1 to a specified number. Two numbers are given. If the current number is divisible by the first number you say "Bosh" instead of that number. If it's divisible by the second you say "Bosh". When the number is divisible by both you say "Bish-Bosh"."Bosh".

For example, if you choose 3 and 5, the output would be:

1, 2, Bish, 4, Bosh, Bish, 7, 8, Bish, Bosh, 11, Bish...

This website was made as an exercise from Lexicon. It lists numbers as if it was playing Bish-Bosh.
